POS IGTF (Impuesto de Grandes Transacciones Financieras) Odoo app helps businesses comply with this tax requirement on large financial transactions in Venezuela. This app is designed to streamline the process of calculating and paying IGTF taxes, also allows businesses to set their own IGTF tax rates based on their specific needs and automatically calculates the IGTF tax for each transaction based on the amount of the transaction and the applicable tax rate.

Customizable IGTF Tax Rates

Allows businesses to set their own IGTF tax rates based on their specific needs.

Automatic IGTF Calculation

Automatically calculates the IGTF tax for each transaction based on the amount of the transaction and the applicable tax rate.

IGTF Journal Configuration

Users need to enable "Is IGTF" option for journal.

IGTF Payment Configuration

User need to select journal with IGTF option and also enable "Is IGTF" option for payment.

POS IGTF Configuration

User need to select POS IGTF Journal and enter IGTF Tax percentage.

If user define IGTF Tax percentage more then 20% it will raise warning.

POS Order

IGTF Payment Method

User can see and pay for pos order using IGTF payment method.

User can see automatically calculates the IGTF tax and total order amount.

POS Receipt

User can also see IGTF tax and total amount in pos order receipt.

IGTF Info in POS Order

User can see "IGTF Info" tab on generated pos order in backend with IGTF information like % IGTF Tax, IGTF Payment Method, IGTF Amount, Total Amount + IGTF, and IGTF Journal.

IGTF Info in Customer Invoice

User can see IGTF information in customer invoice.

IGTF Info in Customer Invoice Report

User can see IGTF information in customer invoice report.
